# 💙 EmoCare: Emotional AI Chatbot for Mental Health Support

## 🏥 Industry
**Healthcare**

## 🎯 Project Overview
EmoCare is an **Emotionally Intelligent Chatbot** designed to detect emotions in user messages and respond empathetically to support mental well-being.  
It uses **TensorFlow** for emotion detection and **Dialogflow** for conversational intelligence.

---

## 🧠 Core Features
- Emotion detection from text (Joy, Sadness, Anger, Fear, Love, Surprise)
- Empathetic chatbot responses
- Interactive dashboard for emotion analytics
- Ethical AI compliance (non-diagnostic, supportive only)

---

## 🛠️ Tech Stack
| Component | Tool |
|------------|------|
| Model Training | TensorFlow / Keras |
| Chatbot Interface | Dialogflow |
| Backend API | Flask |
| Dashboard | HTML + Chart.js |
| Hosting | GitHub Pages |
| Design | Canva / Figma |
